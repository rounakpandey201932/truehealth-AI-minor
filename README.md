# truehealth-AI-minor
AI Mental Wellness Platform

Sign up / Sign in with session persistence via localStorage
Animated orb backgrounds, floating logo, smooth tab transitions

5 question types: emoji scale, slider, multiple choice, open text
Animated progress bar, slide transitions between questions
Analyzing screen with wave animation while Claude processes responses

Real Claude API call analyzes all 10 answers

Dynamic wellness score (10–98) calculated from all answer dimensions
💬 AI Chatbot
Full multi-turn conversation powered by Claude
Knows your name, full check-in summary, and initial analysis
Typing indicator, styled chat bubbles, supportive mental health persona

Home: AI insight, daily quote, personalized recommendations, professional help (if score < 45)
Chat: Full AI chatbot interface
Wellness: 6 interactive activity cards (tap to complete)
Games: Breathing rhythm game + memory matching game
Progress: Mood bar chart, emotional snapshot bars, streak & stats

Mindful Memory: 6 emoji pairs, move counter, win state
